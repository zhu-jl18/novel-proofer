### "调试信息" Tab

```
+---------------------------------------------------------------------+
|  +----------+  +----------+                                         |
|  |   进度   |  | [ 调试 ] |                                         |
|  +----------+  +==========+                                         |
+---------------------------------------------------------------------+
|                                                                     |
|  +-------------------------------------------------------------+   |
|  |  总计: 50  |  完成: 30  |  错误: 2  |  重试: 1               |   |
|  +-------------------------------------------------------------+   |
|                                                                     |
|  过滤: +--------+ +--------+ +--------+                             |
|        | [全部] | | 仅错误 | | 仅重试 |                             |
|        +--------+ +--------+ +--------+                             |
|                                                                     |
|  +-------------------------------------------------------------+   |
|  | #    状态          重试   拆分   错误信息                     |   |
|  +-------------------------------------------------------------+   |
|  | 0    [v] done       0      0                                 |   |
|  | 1    [v] done       0      0                                 |   |
|  | 2    [v] done       1      0                                 |   |
|  | 3    [~] processing 0      0                                 |   |
|  | 4    [ ] pending    0      0                                 |   |
|  | 5    [!] error      3      1     504 Gateway Timeout         |   |
|  | 6    [*] retrying   2      0                                 |   |
|  | 7    [ ] pending    0      0                                 |   |
|  | ...                                                         |   |
|  +-------------------------------------------------------------+   |
|                                                                     |
+---------------------------------------------------------------------+
```

### 状态标记说明

```
[ ] pending    - 灰色背景，待处理
[~] processing - 蓝色背景，处理中（可加动画）
[v] done       - 绿色背景，已完成
[!] error      - 红色背景，错误
[*] retrying   - 橙色背景，重试中
```

## Why
当前的调试/日志信息（chunk 状态列表）直接显示在主页面中，与主要操作区域混在一起，布局不清晰：
- chunk 状态以纯文本 `pre` 块展示，信息密度高但可读性差
- 没有视觉层次区分正常状态、错误状态、重试状态
- 在处理大文件时，大量 chunk 信息会占据页面空间，干扰用户关注核心进度
- 当前 UI 各元素间距不协调，视觉上不够统一

## What Changes
- 采用 Tab 切换方式，将页面分为"进度"和"调试信息"两个 Tab
- "进度" Tab 显示主要操作区域和进度条
- "调试信息" Tab 显示 chunk 状态列表，带视觉区分和过滤功能
- 为不同状态的 chunk 添加视觉区分（颜色标记）
- 添加过滤功能，方便定位特定 chunk 或错误
- **统一调整各元素间距，使整体布局更加协调**

## Impact
- Affected specs: `ui`
